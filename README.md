# Spam Bot

This is a Telegram spam bot.

## Deployment on Railway

1. Fork this repository.
2. Go to [Railway](https://railway.app) and create a new project.
3. Connect your GitHub repository.
4. Set the environment variables in the Railway dashboard:
   - SUDO_USERS
   - BOT_TOKEN
   - BOT_TOKEN2
   - BOT_TOKEN3
   - BOT_TOKEN4
   - BOT_TOKEN5
   - BOT_TOKEN6
   - BOT_TOKEN7
   - BOT_TOKEN8
   - BOT_TOKEN9
   - BOT_TOKEN10
   - MONGO_URI
5. Deploy the project.

## Commands

- .help - Show help menu
- .spam <count> <message> - Spam a message
- .pspam <count> - Spam porn
- .hang <count> - Spam hang message
- .raid <count> <username> - Raid a user
- .ping - Check ping
- .reboot - Reboot the bot
- .sudo <reply> - Add sudo user
- .addsudo <user_id> - Add sudo user by ID
- .showsudo - Show sudo users
- .logs - Show logs
- .echo <reply> - Echo messages
- .rmecho <reply> - Remove echo
- .leave <chat_id> - Leave chat
